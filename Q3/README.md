# Q3 - Contains Duplicate

## Problem
Given an integer array, return true if any value appears at least twice.

## Approach
- Used a hashset to store elements while iterating
- Checked if the element already exists in the set
- If yes → duplicate found

## Time Complexity
O(n)

## Space Complexity
O(n)

## Key Learning
- Learned how hashset helps in fast lookup (O(1))
- Replaced brute force (O(n^2)) with optimized solution
- Understood importance of using sets for duplicate detection
